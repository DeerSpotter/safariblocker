SafariBlocker

Rule Backup Editor

Build whitelist and blocklist backups in your browser, then import the JSON file from the SafariBlocker Settings panel.

Backup format allowedDomains blockedDomains blockedURLs

How to use

  1. Add domains or URLs below.
  2. Click Export Backup JSON.
  3. Move the file to your iPhone if needed.
  4. Open Settings → SafariBlocker → Import List Backup and select the file.
