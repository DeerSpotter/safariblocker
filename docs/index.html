How to use

  1. Import a SafariBlocker backup JSON or add rules below.
  2. Use the + buttons for quick single entries.
  3. Use Batch Paste to paste newline or semicolon separated lists.
  4. Swipe a row left to reveal Delete.
  5. Export the backup JSON and import it from Settings → SafariBlocker → Import List Backup.
